# PolyModel Chat

A powerful, customizable AI chat interface built with React, Vite, and shadcn/ui.

## Features

- **Multi-Model Support**: Access various models via OpenRouter (Llama 3, DeepSeek, etc.).
- **System Prompts**: Configure per-session personas.
- **Custom Models**: Add your own OpenRouter model IDs.
- **Streaming Responses**: Real-time AI generation.
- **Themes**: Multiple built-in color themes (Midnight, Ocean, Sunset, Hacker).
- **Responsive Design**: Works on desktop and mobile.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

1. Clone the repository:

   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:

   ```sh
   cd openchat-pal-main
   ```

3. Install dependencies:

   ```sh
   npm install
   ```

4. Start the development server:
   ```sh
   npm run dev
   ```
